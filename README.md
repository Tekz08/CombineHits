# CombineHits

A World of Warcraft addon for Warriors that tracks and combines multi-hit ability damage into single totals.

## Compatibility
- World of Warcraft Retail (The War Within)

## Features
- Combines multiple hits from abilities into singularly displayed totals.
- Works in AoE, combining all damage done to all targets for supported abilities.
- Movable/lockable display frame.

## Commands
- `/ch` - Show help menu
- `/ch reset` - Reset frame position
- `/ch lock` - Toggle locked frame/transparency
- `/ch debug` - Toggle debug mode (shows detailed logs)

## Installation
Extract to `World of Warcraft/_retail_/Interface/AddOns` and enable in the addon list.
